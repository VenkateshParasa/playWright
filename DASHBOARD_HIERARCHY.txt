Props Flow
==========

Dashboard.tsx
  ├─> mockDashboardData (or API data)
  └─> Distributes to components:

      WelcomeCard(userName, learningTrack, currentDay)
      ProgressOverview(overallProgress, currentWeek, totalWeeks, lessonsCompleted, totalLessons)
      StreakCounter(currentStreak, longestStreak)
      UpcomingReviews(totalDueToday, totalDueTomorrow, upcomingReviews[])
      RecentAchievements(achievements[], totalAchievements)
      StudyTimeChart(data[])
      QuickActions(nextLessonId, nextLessonTitle, reviewsAvailable, exercisesAvailable)


Data Types (dashboard.ts)
==========================

DashboardData
  ├── user: User
  │   ├── name: string
  │   ├── learningTrack: '30-day' | '60-day'
  │   └── currentDay: number
  │
  ├── progress: Progress
  │   ├── overallProgress: number
  │   ├── currentWeek: number
  │   ├── totalWeeks: number
  │   ├── lessonsCompleted: number
  │   └── totalLessons: number
  │
  ├── streak: Streak
  │   ├── currentStreak: number
  │   └── longestStreak: number
  │
  ├── reviews: Reviews
  │   ├── totalDueToday: number
  │   ├── totalDueTomorrow: number
  │   └── upcomingReviews: Review[]
  │       ├── id: string
  │       ├── title: string
  │       ├── dueTime: Date
  │       └── category: string
  │
  ├── achievements: Achievement[]
  │   ├── id: string
  │   ├── title: string
  │   ├── description: string
  │   ├── icon: 'award' | 'trophy' | 'star' | 'zap'
  │   ├── color: string
  │   └── earnedAt: Date
  │
  ├── totalAchievements: number
  │
  ├── studyTime: StudyData[]
  │   ├── day: string
  │   ├── minutes: number
  │   └── lessons: number
  │
  └── quickActions: QuickActionsData
      ├── nextLessonId?: string
      ├── nextLessonTitle?: string
      ├── reviewsAvailable: number
      └── exercisesAvailable: number

Styling Approach
================

All components use:
  - Tailwind CSS utility classes
  - Responsive breakpoints (sm, md, lg, xl)
  - Color-coded themes per component
  - Shadow and border utilities
  - Hover and transition effects
  - Rounded corners and padding
  - Grid and flexbox layouts


State Management
================

Dashboard.tsx:
  - useState for loading state
  - useState for dashboard data
  - useEffect for data fetching
  - Conditional rendering based on loading

Components:
  - Pure presentational components
  - Props-driven rendering
  - No internal state management
  - Optional loading props for individual skeletons
