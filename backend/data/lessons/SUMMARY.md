## JSON Schema

Each lesson follows a consistent schema with:

```
- id, title, slug (identification)
- track, category, order (organization)
- duration, estimatedXP (metrics)
- description, objectives (learning goals)
- prerequisites, nextLesson (progression)
- content sections (educational material)
- keyTakeaways (summary)
- resources (external links)
- tags (searchability)
```
