## Discussion Forums

### Categories

The forum is organized into four main categories:

- **General**: General discussions about Playwright, Selenium, and testing
- **Help**: Ask questions and get help from the community
- **Show & Tell**: Share your projects, achievements, and success stories
- **Announcements**: Official announcements and updates

### Creating a Thread

1. Navigate to the Community Forum
2. Click "New Thread"
3. Choose a category
4. Write a descriptive title (required)
5. Add content using the rich text editor
6. Add tags to help others find your thread
7. Preview and submit

### Rich Text Features

- **Markdown support**: Use markdown syntax for formatting
- **Code syntax highlighting**: Use triple backticks with language identifier
- **Images**: Upload images directly or use image URLs
- **@mentions**: Mention users with @username to notify them
- **Links**: Add hyperlinks to external resources

### Voting System

- Upvote helpful or interesting threads/replies
- Downvote low-quality or off-topic content
- Vote score = Upvotes - Downvotes
- Helps surface quality content

### Best Answer

Thread authors can mark the most helpful reply as the "Best Answer":
- Helps others find solutions quickly
- Rewards helpful community members with reputation points
- Marked replies appear at the top

### Bookmarking

- Bookmark threads to save them for later
- Access bookmarked threads from your profile
- Receive notifications when bookmarked threads have new activity

### Search and Filters

- Search by keywords in titles and content
- Filter by category
- Sort by:
  - Recent: Newest activity first
  - Popular: Most views and upvotes
  - Unanswered: Threads without replies
  - Trending: Hot discussions from the past 3 days

## Study Groups

### Creating a Study Group

1. Go to Study Groups section
2. Click "Create Group"
3. Fill in details:
   - Group name
   - Description
   - Category (Playwright, Selenium, General Testing, etc.)
   - Tags
   - Privacy (Public or Private)
   - Max members limit
   - Study schedule (optional)
   - Goals

4. Submit to create

### Group Features

#### Membership Roles

- **Owner**: Full control over the group
- **Moderator**: Can manage content and invite members
- **Member**: Can participate in discussions and challenges

#### Group Chat/Discussion

- Dedicated discussion board for group members
- Post announcements, questions, and updates
- Share resources and materials

#### Shared Progress Tracking

- See aggregated progress of all members
- View individual member milestones
- Celebrate achievements together

#### Group Challenges

- Create time-bound learning challenges
- Set point values for completion
- Track participant progress
- Display challenge leaderboard

#### Study Schedule

- Coordinate study sessions
- Set recurring meeting times
- Specify timezone
- Send reminders to members

#### Resources Sharing

- Share links, files, and documents
- Organize learning materials
- Curate helpful resources for the group

#### Group Leaderboard

- Rank members by reputation
- Display lessons and quizzes completed
- Show helpful contributions
- Encourage friendly competition

### Joining Groups

**Public Groups:**
- Browse and join directly
- No approval needed

**Private Groups:**
- Require invite code
- Request invitation from members
- Owner/moderators approve requests

### Leaving Groups

- Members can leave at any time
- Owners must transfer ownership before leaving
- Moderator status is removed when leaving

Earn reputation points by:
- Completing lessons: 10 points
- Passing quizzes: 15 points
- Creating quality threads: 5 points
- Posting helpful replies: 3 points
- Best answer marked: 10 points

- Helping others: 5 points
