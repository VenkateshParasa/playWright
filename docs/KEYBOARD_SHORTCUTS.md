## Tips for Keyboard Users

### General Tips

1. **Use Tab for navigation** - Tab moves forward, Shift+Tab moves backward
2. **Use Enter to activate** - Press Enter on focused links and buttons
3. **Use Space for toggles** - Press Space on checkboxes and toggles
4. **Use Arrow keys in widgets** - Navigate menus, dropdowns, and custom components
5. **Use Escape to exit** - Close dialogs, cancel actions, exit modes
6. **Look for focus indicators** - Blue outline shows where you are

### Efficiency Tips

1. **Learn skip links** - Press Tab on page load to skip navigation
2. **Use keyboard shortcuts** - Much faster than tabbing to buttons
3. **Use search** - Ctrl/Cmd+K to quickly find anything
4. **Learn landmarks** - If using screen reader, jump between sections
5. **Use the shortcuts help** - Press ? to remind yourself of shortcuts

### Screen Reader Tips

1. **Learn your SR shortcuts** - Each screen reader has different commands
2. **Use landmark navigation** - Jump to main, nav, search quickly
3. **Use heading navigation** - Press H to jump between headings
4. **Use form mode** - Switch to forms mode when filling out forms
5. **Listen to page structure** - Get page summary before diving in

## Customization

### Creating Custom Shortcuts

Some browsers and screen readers allow custom keyboard shortcuts:

1. **Browser Extensions** - Install keyboard navigation extensions
2. **Screen Reader Settings** - Configure custom commands
3. **User Scripts** - Advanced users can create custom scripts
4. **OS Shortcuts** - Set up system-wide shortcuts

### Disabling Shortcuts

If any shortcut conflicts with assistive technology:

1. Navigate to **Settings > Preferences**
2. Select **Keyboard Shortcuts**
3. Disable or remap conflicting shortcuts
